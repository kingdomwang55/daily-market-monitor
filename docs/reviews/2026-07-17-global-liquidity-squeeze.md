**核心结论**：这不是资金搬家的下跌，是**美元现金虹吸的下跌**。钱去了短端美债和货币基金，同时保证金 call 逼平了黄金和股票。

## 2. 钱去哪了？三个答案

### 2.1 短端美元 / 货币基金 / T-bill（最主要）

**证据**：美债 10Y、2Y 收益率在 4.5%+ 高位站着。传统避险剧本是"股跌 → 债涨 → 收益率下行"，这次没发生。说明避险资金**没去长债**，而是去了：
- 隔夜逆回购（RRP）
- 短端 T-bill（3M 现在 4%+ 无风险）
- 美元现金存款

这是"流动性收缩型"下跌的典型标志 — 大家在**囤美元现金**，不是买风险对冲工具。

### 2.2 原油 / 通胀敏感型商品（少数）

**证据**：WTI 4 天涨 13% 到 80.70。这不是需求驱动（需求驱动会跟股市同涨同跌），是**地缘/供给冲击 + 滞胀交易**。有一部分对冲基金在做多"通胀不下、增长下杀"的滞胀组合。

### 2.3 强制平仓 / Margin Call 卖出（隐性去处）

**证据**：黄金 4 天跌 4.5%，跌得比股票还猛。黄金前期从 3800 涨到 4148 是拥挤交易（COT 净多头历史高位）。**当股票和债券同时亏损时，杠杆账户会被 margin call 卖掉最"好卖"的东西 — 就是黄金**。

这是 **2020 年 3 月疫情初期**和 **2008 年 9 月雷曼**都出现过的经典桥段：**黄金短期被杀，之后大幅反弹**。

## 3. 为什么"黄金不避险"？

三个原因叠加：

1. **实际利率跳升**：名义 10Y 4.57% + 通胀预期没跌 = 实际利率上行 → 黄金分母端受压
2. **前期太拥挤**：4148 是历史高点，获利盘 + 杠杆盘密集
3. **保证金 call**：卖别的资产不好卖，卖黄金能立刻变现

黄金**不是失去避险属性**，是**避险功能在流动性事件中会暂时失灵**。之后如果美联储放水或 Fed put 出现，黄金会第一个反弹。

## 4. 这是什么剧本？

**滞胀 + 流动性收缩双杀**（Stagflation + Liquidity Squeeze）：

- **分子端**：经济数据走弱 → 股票盈利预期下修
- **分母端**：利率高位不下 → 估值杀
- **现金端**：美元虹吸 → 一切资产被抽血
- **通胀端**：原油涨 → 央行不敢降息

三个通道同时收紧，唯一赢家是**美元现金**和**短端美债**。
