# 预警规则详解

## 分级预警

| 级别 | 阈值 | 含义 | 触发动作 |
|---|---|---|---|
| ⚪ 无 | < 1.5% | 平稳 | 不推送 |
| ⚠️ 注意 | ≥ 1.5% | 轻度波动 | 推送指数信息 |
| ⚠️⚠️ 警戒 | ≥ 2.5% | 中级异动 | 推送 + 拉板块数据 |
| 🚨🚨🚨 严重 | ≥ 3.5% | 严重异动 | 推送 + 板块 + 建议 |

## 各监控具体规则

### stabilize（企稳信号）
命中 5 类信号中 **≥ 2 个**触发推送：

| 信号 | 判断条件 |
|---|---|
| ✅ 止跌确认 | 最新低点 ≥ 前日低点 × 0.995 且 跌幅 < 1% |
| ✅ 均线修复 | 收盘 ≥ MA5 |
| ✅ 阳线反包 | 收盘 > 前日开盘 且 收 > 开 |
| ✅ 缩量止跌 | 成交量 < 前日 × 0.9 |
| ✅ 锤头形态 | 下影线 > 实体 × 2 且 阳线 |

每日首次运行附带 **防御性资产快照**。

### us_market（美股夜盘）
- 指数（道琼斯/纳指/标普）：波动 ≥ **±1.5%** 推送
- 个股（拼多多/英伟达等 10 只）：波动 ≥ **±3.0%** 推送
- Marker 按整数点位分级（避免同一档反复推送）
- 每次定时任务都会检查一次，触发才推

### hk_market（港股）
- 指数（恒指/恒生科技/恒生国企）：波动 ≥ **±1.5%** 推送
- 个股（腾讯/美团等 8 只）：波动 ≥ **±3.0%** 推送
- 快照模式：09:32 开盘 / 12:00 午间 / 16:05 收盘

### shock（A 股异动）
- 大盘指数分级预警：1.5% / 2.5% / 3.5%
- 板块异动阈值：**±4.0%**（只在指数 L2+ 时才拉板块数据）
- 频率：**每 10 分钟**
- 每整数级别每日只推一次

### price_alert（关键点位）
- 上证指数：止损 **3907** / 加仓 **4318**
- 创业板指：止损 **4048** / 加仓 **4474**
- 沪金主力：止损 **844** / 加仓 **932**
- 频率：每 30 分钟
- 首次跌破/突破触发，回到区间后清 marker 可再次触发

## Marker 机制

**为什么要 Marker？**
避免同一个点位被反复推送打扰。

**Marker 生命周期**：
1. 触发 → 写入 state → 推送
2. 回到区间（>阈值 × 1.01 或 <阈值 × 0.99）→ 清除 marker
3. 再次触发 → 重新推送

**Marker 存储**：`/tmp/{monitor_name}_state.json`
