| **VIX/SKEW** | 恐慌区 | 未恐慌但 SKEW 抬升 | 尚未底部情绪 |

### 3.2 为什么差异如此重要

**3 月那次的成立条件**：
1. 年线是**中期趋势线**——3 月还没破，意味着中期上升结构完好
2. 放量恐慌意味着**卖盘释放充分**，之后没人再有子弹继续砸
3. 单日急跌 + 长下影线 = 底部特征明显

**今天的问题**：
1. 年线已破，中期结构进入**下降趋势**
2. 缩量阴跌意味着**没有恐慌盘释放**，卖盘还在源源不断（钝刀割肉）
3. 估值分位仍在高位 = **盈利端在恶化**，不是单纯情绪杀
4. A 股缩量跌 vs 港股逆势涨 = **国内资金/情绪问题**，非外部冲击

## 4. 反直觉发现

### 4.1 估值分位与点位倒挂
- 上证 380 PE 分位 **91.7%**、沪深 300 PE 分位 **90.9%**
- 但指数已从高点回撤 ~7%
- **含义**：EPS 下滑速度 > 股价下跌速度
- **结论**：跌了反而更贵。这不是估值底。

### 4.2 缩量不是企稳，是钝刀割肉
- 今日成交 535 亿手 vs 3 月恐慌日 804 亿手（-37%）
- 没有放量恐慌 = 没有强制平仓盘 = 没有错杀 = 没有反转动能
- **卖压来自持续赎回/减仓，不是集中抛售**

### 4.3 汇率稳定是假安全感
- USDCNH 6.7680 表面平稳
- SKEW 148.51 显示尾部风险溢价抬升
- **市场在买保险，只是还没出事**

### 4.4 A/H 分化异常
- 港股主要科技股全部大涨 +2% ~ +6%
- A 股同板块（科创、创业板）大跌 -3% ~ -4%
- 说明：**跌的不是"中国资产"，跌的是"A 股本身"**（流动性 / 情绪 / 结构问题）

## 5. 什么条件下可以入场？

不是不能抄，是**现在不到时候**。以下任一条件出现再考虑：

### 5.1 技术条件（任一）
- [ ] 出现**单日放量长下影**（成交量 > 800 亿手 + 跌幅 > 2%）
- [ ] 上证跌破 3800 后**快速收复站上 3800**（V 型反转）
- [ ] 距 MA20 乖离 > -6%（今天只有 -3.8%）
- [ ] 出现连续两日的**放量止跌**（不是缩量）

### 5.2 估值条件（任一）
- [ ] 沪深 300 PE 分位从 90.9% 快速回落至 < 60%
- [ ] 出现单日 EPS 一致预期上修（说明盈利拐点）

### 5.3 情绪条件（任一）
- [ ] SKEW > 155 + VIX 快速抬升（恐慌见顶特征）
- [ ] 融资余额单日缩水 > 200 亿（去杠杆完成）
- [ ] 出现明显的政策底/救市信号

### 5.4 分批策略（如果一定要参与）
